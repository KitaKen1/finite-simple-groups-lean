# M22 (and PSL(3,4))

This document records the minimum provenance and certificate policy for the
Lean formalization of Mathieu's group `M22` and of the projective group
`PSL(3,4)` used as its point stabilizer.

## Lean entry points

```lean
import FiniteSimpleGroups.Sporadic.M22

#check Sporadic.M22
#check Sporadic.card_M22
#check Sporadic.simple_M22
#check Sporadic.L34.simple_L34          -- PSL(3,4) is simple
#check Sporadic.M22PointStabilizer.card_L34  -- |PSL(3,4)| = 20160
```

The public theorems are:

```lean
theorem Sporadic.card_M22 : Nat.card Sporadic.M22 = 443520
theorem Sporadic.simple_M22 : IsSimpleGroup Sporadic.M22
```

## Definition and provenance

Unlike `M11`/`M12`, no generator data was taken from the literature.
An external Python construction builds everything from first principles:
PG(2,4) (21 points, 21 lines over GF(4)); `PSL(3,4)` from eight elementary
transvections (order 20160 verified by BFS); the Steiner system `S(3,6,22)`
as the 21 sets `line ∪ {∞}` plus one `PSL(3,4)`-orbit of 56 hyperovals
(Steiner property checked exhaustively); a design automorphism moving `∞`
found by backtracking; and finally a verified 2-element generating set
(`m22a` of order 5, `m22b` of order 8) for the resulting group of order
443520.

## Certificate policy

All candidate data was generated by external Python scripts and is
re-verified inside Lean by kernel-checked `decide` proofs (some heavy
certificates use `decide +kernel`, which is still kernel-checked reduction
and adds no axioms).

* `Nat.card M22 = 443520` follows from the verified stabilizer chain
  `22·21·20·16·3`. `M22` is only 3-transitive: the upper bounds at the last
  two levels come from setwise preservation of the unique `S(3,6,22)` block
  through the first three base points. The five-point stabilizer step splits
  on the image of one point: one branch propagates to the identity, the
  other to the odd outer involution of `Aut(S(3,6,22)) = M22:2`, excluded by
  the parity certificate `M22Subgroup ≤ alternatingGroup (Fin 22)`.
* `PSL(3,4)` (the `L34/` layer, on `Fin 21`) is generated by three explicit
  elations. Certificates: the full 21×15 elation family with conjugation
  index tables for the generators (extended to the whole group by closure
  induction), commutativity of the elations at the base point, membership
  words, and commutator identities expressing each generator as `⁅u,v⁆` of
  elations. A self-contained Iwasawa-style argument then shows every
  nontrivial normal subgroup contains the (full) commutator subgroup.
  The order `20160 = 420·48` is pinned by an orbit-stabilizer lower bound
  (48 distinct two-point-stabilizer words) against the embedding upper
  bound into the `M22` point stabilizer (of order `443520/22`).
* `IsSimpleGroup M22` is the classical point-stabilizer argument: the
  stabilizer of the last point is the embedded `PSL(3,4)`; a nontrivial
  normal subgroup is transitive (two-transitivity word certificates), and
  the regular case of order 22 is excluded by Cauchy's theorem
  (orders 2 and 11 cannot coincide).
